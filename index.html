Turrisclock

Driving clock movement from Turris Router

Download this project as a .zip file Download this project as a tar.gz file

This is a hardware and software design allowing you to connect common quartz analog clock movement to Turris router and let it controlled by Turris router. Such clock features some ultimate features:

  • precise atomic time thanks to NTP
  • ability to pause and resume movement at any moment (no more ticking disturbing your sleep)
  • battery-free operation
  • special effect like stepping 5 seconds at time
